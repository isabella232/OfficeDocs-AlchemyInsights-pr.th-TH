# <a name="keyboard-settings-in-windows-10"></a>การตั้งค่าแป้นพิมพ์ใน Windows 10

**การตั้งค่าชนิด**

เมื่อต้องการเปลี่ยนการตั้งค่า Windows ที่เกี่ยวข้องกับการตรวจสอบการสะกด การแก้ไขอัตโนมัติ ข้อเสนอแนะ และลักษณะอื่นๆ ของฮาร์ดแวร์และแป้นพิมพ์สัมผัสของคุณ ให้ไปที่ การตั้งค่า >**อุปกรณ์> การพิมพ์** 

**การตั้งค่าภาษาและคีย์บอร์ด**

เมื่อต้องการเปลี่ยนการตั้งค่าที่เกี่ยวข้องกับคีย์บอร์ดหรือวิธีการป้อนข้อมูลที่จะใช้กับภาษา Windows ให้ไปที่ การตั้งค่า >**ภาษา&ภาษา>ภาษา** เลือกภาษาในรายการ **ภาษาที่ต้องการ** แล้วคลิก ตัวเลือก **เพื่อเพิ่ม** /เอาคีย์บอร์ดหรือวิธีการป้อนข้อมูล (IMEs) ออก
