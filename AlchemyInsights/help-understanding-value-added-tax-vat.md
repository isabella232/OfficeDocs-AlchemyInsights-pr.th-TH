# <a name="help-understanding-value-added-tax-vat"></a>ช่วยให้เข้าใจภาษีมูลค่าเพิ่ม (VAT)

ถ้าคุณอยู่ในประเทศที่ใช้ภาษีมูลค่าเพิ่ม (VAT) เช่น สหราชอาณาจักร Microsoft Invoicing จะคํานวณ VAT ให้คุณ และใช้จํานวนดังกล่าวกับใบแจ้งหนี้ของคุณ
  
ด้วยเหตุผลด้านภาษี คุณยังต้องระบุหมายเลขจดทะเบียน VAT ของคุณในการตั้งค่า Invoicing
  
คุณไม่สามารถเพิ่มหรือลบอัตรา VAT ออก แต่คุณสามารถเปลี่ยนอัตรา VAT เริ่มต้นการตั้งค่าได้ ตามค่าเริ่มต้น Invoicing จะใช้บริการเพื่อตรวจสอบหมายเลขจดทะเบียน VAT แต่เราขอแนะนนะให้คุณอ่านข้อสงวน [สิทธิ์ VIES](https://go.microsoft.com/fwlink/?LinkID=841741) ที่เผยแพร่โดยคณะกรรมาธิการยุโรป ถ้าคุณเลือกที่จะปิดใช้งานบริการ คุณต้องตรวจสอบหมายเลขจดทะเบียน VAT ทั้งหมด
