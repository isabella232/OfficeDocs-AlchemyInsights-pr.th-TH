# <a name="single-sign-on-sso-in-microsoft-edge"></a>การลงชื่อเข้าระบบครั้งเดียว (SSO) ใน Microsoft Edge

Microsoft Edge สนับสนุนการลงชื่อเข้าระบบครั้งเดียว (SSO) ชนิดต่อไปนี้:
- SSO ที่ใช้ PRT
- SSO อย่างราบรื่น
- SSO แบบใช้ WIA
