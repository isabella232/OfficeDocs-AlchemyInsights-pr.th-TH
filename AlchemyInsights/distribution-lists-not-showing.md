# <a name="distribution-group-not-showing-in-groups-list"></a>กลุ่มการแจกจ่ายไม่แสดงในรายการกลุ่ม

อาจใช้เวลาถึง 60 นาทีก่อนรายชื่อการแจกจ่ายใหม่ของคุณจะปรากฏในศูนย์การจัดการ Exchange
  
ถ้าคุณต้องการเพิ่มสมาชิกหรือแก้ไขอื่นๆ ในตอนนี้ โปรดใช้ฟีเจอร์ กลุ่ม ใน[ศูนย์การจัดการ Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)
