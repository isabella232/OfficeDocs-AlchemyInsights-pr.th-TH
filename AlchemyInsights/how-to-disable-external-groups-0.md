# <a name="how-to-give-access-to-external-users-in-yammer"></a>วิธีมอบสิทธิ์การเข้าถึงให้กับผู้ใช้ภายนอกในYammer

มีสี่วิธีที่ผู้ใช้ภายนอกสามารถเข้าถึงเครือข่ายYammerได้
  
1. **ผู้ใช้ที่เป็น** Yammer - ผู้ดูแลระบบสามารถเชิญที่ติดต่อจากโดเมนภายนอกเป็นผู้ใช้บนเครือข่ายหลักได้ ผู้ใช้เหล่านี้จะเห็นตัวดึงข้อมูล All Company และสามารถเข้าร่วมกลุ่มอื่นๆ ในเครือข่ายได้

2. **ผู้ใช้ภายนอกในกลุ่มภายนอก** - กลุ่มภายนอกอนุญาตให้ผู้ใช้ภายนอกได้ ขอบเขตของการเข้าถึงของผู้ใช้ภายนอกจะถูกจํากัดไว้เฉพาะกลุ่มภายนอก ด้วยกลุ่มภายนอก คุณสามารถเชิญที่ติดต่อจากYammerที่อนุญาตให้ผู้ใช้เข้าร่วมการสนทนาภายนอกเท่านั้น

3. **ผู้ใช้ภายนอกในYammerการสนทนาในกลุ่มภายใน**- ตัวเลือกนี้อนุญาตให้สามารถเพิ่มผู้เข้าร่วมภายนอกลงในการสนทนาเฉพาะได้ ผู้เข้าร่วมภายนอกสามารถเข้าร่วมการสนทนาที่พวกเขาได้รับเชิญอย่างชัดเจนและเข้าร่วมสิ่งที่แนบมาในการสนทนานั้นเท่านั้น พวกเขาเข้าถึงการสนทนาเหล่านี้ผ่านทางYammerของพวกเขา (บนเครือข่ายของพวกเขา) พวกเขาไม่สามารถเข้าถึงส่วนที่เหลือของเครือข่ายของคุณ

4. **เครือข่ายภายนอก** - ตัวเลือกนี้ช่วยให้คุณสามารถเชิญผู้ใช้ภายนอกไปยังเครือข่ายภายนอกที่ผู้ดูแลระบบที่ผ่านการตรวจสอบสามารถสร้างในเครือข่ายของคุณ ตัวเลือกนี้เหมาะที่สุดกับการโฮสต์ชุมชนภายนอก

For more information, please see [Add external messaging participants to your Yammer conversations](https://docs.microsoft.com/yammer/work-with-external-users/add-external-participants) and [External Messaging FAQ (Yammer)](https://docs.microsoft.com/yammer/work-with-external-users/external-messaging-faq)
