# <a name="about-briefing-email"></a>เกี่ยวกับอีเมลฉบับย่อ

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับ การบรรยายสั้นๆ [ให้ดู ภาพรวมของอีเมลฉบับ](https://docs.microsoft.com/briefing/be-overview)ย่อ  

ขณะนี้ฟีเจอร์นี้จะถูกเปิดให้ใช้งาน ดังนั้นองค์กรของคุณอาจไม่ได้รับอีเมลฉบับย่อ แม้ว่าคุณจะเปิดใช้งาน การบรรยายสั้นๆ ไว้ นอกจากนี้ โปรดทราบว่าผู้ใช้ไม่รับประกันว่าจะรับอีเมลฉบับย่อทุกวัน อีเมลจะถูกส่งเฉพาะเมื่อผู้ใช้มีอย่างน้อยหนึ่งข้อเสนอแนะที่สามารถปฏิบัติได้ในวันนั้น For more info, see [Frequently asked questions for the Briefing email](https://docs.microsoft.com/briefing/be-faqs).
