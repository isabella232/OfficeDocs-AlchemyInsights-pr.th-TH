# <a name="view-and-delete-browsing-history-in-microsoft-edge"></a>ดูและลบประวัติการเรียกดูในMicrosoft Edge

ด้วยการอนุญาตของคุณ Microsoft Edgeจะบันทึกข้อมูลการเรียกดูของคุณ ซึ่งจะช่วยให้คุณกลับไปยังไซต์โปรดหรือกรอกแบบฟอร์ม ในโหมด InPrivate, Guest และโหมดMicrosoft Edge จะจัดเก็บข้อมูลน้อยกว่าที่จัดเก็บระหว่างการเรียกดูปกติ

ข้อมูลการเรียกดูจะถูกจัดเก็บบนอุปกรณ์ของคุณ ถ้าคุณเปิดการซิงค์ ข้อมูลจะถูกจัดเก็บไว้ในระบบคลาวด์และซิงค์กับอุปกรณ์ต่างๆ ที่คุณลงชื่อเข้าใช้บัญชี Microsoft Edgeของคุณ

เมื่อต้องการดูประวัติการเรียกดู ให้เลือก การตั้งค่า **และอื่นๆ ...**   >  **ประวัติ**  >  **จัดการ** ประวัติ คุณสามารถล้างประวัติการเรียกดูของคุณได้ตลอดเวลา
