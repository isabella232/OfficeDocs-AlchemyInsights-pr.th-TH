# <a name="user-mailbox-creation-is-taking-a-long-time"></a>การสร้างกล่องจดหมายของผู้ใช้ใช้เวลานาน

อาจใช้เวลาถึง 24 ชั่วโมงก่อนที่กล่องจดหมายใหม่จะพร้อมให้ใช้งาน
  
ถ้ากล่องจดหมายของผู้ใช้ "การเตรียม" นานกว่าหนึ่งชั่วโมง ให้ลองเอาสิทธิ์การใช้งาน Exchange ออกจากผู้ใช้แล้วเพิ่มกลับอีกครั้ง หรือ[ติดต่อ](/microsoft-365/admin/contact-support-for-business-products?tabs=online)ฝ่ายสนับสนุน
