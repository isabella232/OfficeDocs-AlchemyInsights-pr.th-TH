# <a name="dashboard-not-loading"></a>แดชบอร์ดไม่โหลด

**อาการ:**

แดชบอร์ดอาจติดอยู่กับตัวหมุนการโหลด หรืออาจแสดงข้อความแสดงข้อผิดพลาดทั่วไป: "มีบางอย่างผิดพลาด ตรวจสอบรายละเอียดด้านเทคนิคเพื่อดูรายละเอียดเพิ่มเติม"
