# <a name="cancel-your-microsoft-subscription"></a>ยกเลิกการสมัครใช้งาน Microsoft ของคุณ

ผู้ดูแลระบบสามารถยกเลิกการสมัครใช้งานขององค์กรของพวกเขาศูนย์การจัดการ Microsoft 365ได้

1. ไปที่หน้า \> [การเรียกเก็บเงิน ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)ของคุณ
2. **บนแท็บ** ผลิตภัณฑ์ ให้ค้นหาการสมัครใช้งานที่คุณต้องการยกเลิก เลือกจุดสามจุด (การทําเพิ่มเติม) **จากนั้นเลือก ยกเลิก** การสมัครใช้งาน
3. ในบานหน้าต่าง **ยกเลิก** การสมัครใช้งาน ให้เลือกเหตุผลว่าเหตุใดคุณจึงยกเลิก อีกทางหนึ่งคือ ให้ข้อเสนอแนะใดๆ
4. เลือกบันทึก

**การคืนเงิน:** เครดิตตามส่วนใดๆ จะถูกส่งกลับถึงคุณภายในรอบการเรียกเก็บเงินถัดไป
