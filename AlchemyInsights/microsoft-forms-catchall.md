# <a name="get-information-about-microsoft-forms"></a>การรับข้อมูลเกี่ยวกับ Microsoft Forms

สำหรับข้อมูลเฉพาะเกี่ยวกับ Microsoft Forms ให้บรรยายคำถามของคุณเพื่ออธิบายปัญหาของคุณรวมถึงข้อผิดพลาดฟีเจอร์ที่คุณกำลังใช้เป็นต้น 

ต่อไปนี้คือแหล่งข้อมูลเพิ่มเติมบางอย่าง:

- [ตรวจสอบการเปิดใช้งานสิทธิ์การใช้งานและสถานะของ Azure CollabDBService สำหรับผู้เช่าของคุณ](https://support.office.com/article/Turn-off-or-turn-on-Microsoft-Forms-8dcbf3ab-f2d6-459a-b8be-8d9892132a43)
- [แชร์แบบทดสอบหรือฟอร์มใน Microsoft Forms](https://support.office.com/article/Share-a-form-to-collaborate-d5bb5cf0-8401-4c15-bb8c-8e108cd7e69b)
- [จัดการความสามารถในการแชร์ฟอร์มภายนอก](https://support.office.com/article/set-up-microsoft-forms-cc52287a-4550-464d-9a1b-457bf9df2240?#PickTab=Configure)
- [ลบฟอร์มหรือกู้คืนฟอร์มที่ถูกลบ](https://support.office.com/article/Delete-a-form-2207e468-ce1b-4c4a-a256-caf631d87af0)
- [พิมพ์ผลลัพธ์ในรูปแบบของข้อมูลสรุปของการตอบกลับ](https://support.office.com/article/Print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc)
- [ย้ายแบบสำรวจแบบทดสอบหรือการสำรวจความคิดเห็นไปยังกลุ่ม](https://support.office.com/article/Transfer-ownership-of-a-form-921a6361-a4e5-44ea-bce9-c4ed63aa54b4)
- [โอนความเป็นเจ้าของของฟอร์มไปยังผู้ใช้อื่น](https://support.office.com/article/Transfer-ownership-of-a-form-921a6361-a4e5-44ea-bce9-c4ed63aa54b4)
