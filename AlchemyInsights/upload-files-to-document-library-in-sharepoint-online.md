# <a name="upload-files-to-onedrive-or-sharepoint"></a>Uploadไฟล์OneDriveหรือSharePoint

คุณสามารถอัปโหลดไฟล์ไปยังไลบรารีเอกสารใน SharePoint Online ได้เพียงแค่ลากไฟล์เหล่านั้นจากคอมพิวเตอร์ของคุณแล้วปล่อยลงในไลบรารีเอกสาร ด้วยOneDriveคุณสามารถซิงค์ไฟล์ระหว่างคอมพิวเตอร์ของคุณและระบบคลาวด์เพื่อให้คุณสามารถเข้าถึงไฟล์ของคุณได้จากทุกที่ คุณยังสามารถใช้งานไฟล์ที่คุณซิงค์ได้โดยตรงใน File Explorer และเข้าถึงไฟล์ของคุณแม้ในขณะออฟไลน์ หากต้องการข้อมูลเพิ่มเติมเกี่ยวกับการอัปโหลดและซิงค์ โปรดไปที่ลิงก์ด้านล่าง
